Guide to install and run

What IS settings-fixer?

This is a software made for die-heart Minecraft fans. Many people are out there who play different versions of minecraft java for example someone may play the latest versions for survival and 1.12 for mods or 1.8.9 for pvp. So while switching between the versions ,the controls get randomly reseted which is actually very frustrating thing to fix. So I developed this programme which is programmed in "C" programming language and use CMD commands to backup your minecraft controls which can be reverted in only one click. This programme is supported by all minecraft clients like lunar, badlion , original minecraft launcher, batmod , labymod , tlauncher or any other client which use .minecraft directory as root directory. This also supports Salwyrr minecraft launcher which uses .Salwyrr directory as its root directory.
